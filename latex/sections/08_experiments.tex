表~\ref{tab:a2-main} 揭示了几个核心发现: (1) \textbf{Real EEG 在所有六种退化条件下均优于 vision-only}, 提升幅度在 +0.0252（clean）到 +0.3255（mixed）之间. 退化越严重, EEG 的增益越大, 这与``视觉信息不足时大脑额外信息更有用''的假设一致. (2) \textbf{Real EEG 在所有条件下均大幅优于 shuffled EEG 和 random EEG}, Real-Shuffled 增益为 +0.3457 到 +0.6496, Real-Random 增益为 +0.2943 到 +0.5638. Shuffled EEG 和 random EEG 的表现始终在机会水平附近（对 40 类任务, 随机概率为 0.025), 说明模型学到的 EEG 信息确实来自真实配对而非容量增益. (3) \textbf{Mixed 退化条件下 gap 最大}, 因为复合退化对视觉特征破坏最严重, EEG 语义信息起了最大作用. (4) \textbf{Win rate = 18/18}: 在 3 种子 × 6 条件的全部组合中, real EEG 均优于 vision-only 和 shuffled/random 对照.

\subsection{A2 与其他语义融合方法对比}

本文在语义预测任务上还将 A2 与以下方法进行了对比:

\begin{itemize}
  \item \textbf{P2 (Raw+Spectrogram Fusion):} 将原始时序特征和频谱特征直接拼接融合, 无双分支交互;
  \item \textbf{P2A2:} P2 encoder + A2 风格的 three-branch 后融合;
  \item \textbf{Residual Adapter:} 在视觉嵌入上加入 EEG-based residual adapter, 无门控;
  \item \textbf{Prototype Bias:} 在文本原型相似度上加入 EEG 预测的偏移项;
  \item \textbf{Combined:} Gated fusion + prototype bias 的组合方案.
\end{itemize}

A2 最终成为最强定量模型的原因是: three-branch temporal-spectral-spatial encoding 提供了最丰富的信息源; gated residual fusion 的输入自适应特性防止了 EEG 在高质量图像上的过度干预; 且 A2 的 transformer-based 后融合使各支路特征得到充分交互.

\subsection{VTF Token-level 结果}

VTF 在 token-level 成功实现了 real EEG 优于 vision-only 和 EEG 对照, 证明 token-level EEG fusion 是可行的. 但 VTF 的 absolute classification accuracy 低于 A2 的 pooled fusion. 例如在 lowres16 条件下, VTF 的 top-1 accuracy 约为 0.18（real EEG）, 而 A2 为 0.52. 这一差距源于: A2 的融合网络更深更复杂, 而 VTF 仅用一次 cross-attention.

关键发现: (1) \textbf{Route5 全面优于 Route1}, 说明原生视觉语言模型对生成式任务的重要性远大于纯文本 LLM; (2) \textbf{LoRA r=8 在 class-hit 上优于 r=16}, 验证了小样本下低秩约束的正则化效应; (3) \textbf{best-of-N reranking 显著提升了 valid caption rate} (从 91.89\% 提升到 97.75\%), 且维持了较高的 class-hit; (4) \textbf{所有 Route5 变体在 real EEG 下的 class-hit 均高于 vision-only 和 shuffled/random EEG}, real-vision gap 为 +8.41\%--+9.10\%, real-shuffled gap 为 +10.32\%--+11.05\%.

然而, 生成式 EVLM 的 absolute class-hit 仍远低于 A2 的 top-1 accuracy. 这表明生成式模型的主要价值在于产生自然 language caption 用于展示, 而非取代 A2 的分类能力.

\subsection{Caption Target 与 LoRA 秩消融}

\textbf{Caption target 对比}: T1 (class-only target) 在 strong 退化条件下的 real EEG class-hit 为 0.4397, valid caption rate 为 0.9374. T3 (class+BLIP caption) 在同条件下 real EEG class-hit 为 0.3525, valid caption rate 为 0.9795. T1 的 class-hit 更高但 caption 形式是类别名而非自然语言; T3 生成的 caption 更自然但 class-hit 稍低. 本文以 T3 作为主展示策略, 因为其 natural-language output 更符合生成式 EVLM 的定义.

\textbf{LoRA 秩消融}: $r=8$ 的 strong real class-hit 为 0.3826, $r=16$ 为 0.3040. $r=8$ 的 valid caption rate 为 0.9189, $r=16$ 为 0.8363. 两个指标都表明 $r=8$ 优于 $r=16$, 可能是因为小训练集下可训练参数过大导致过拟合.

\subsection{实验讨论}

综合所有实验结果, 本文可以得到以下结论:

\begin{enumerate}
  \item \textbf{A2 是当前设定下的最强定量 semantic model}. 它在 18/18 的 seed-condition 组合中 real EEG 均优于所有对照. A2 的成功归因于 three-branch temporal-spectral-spatial 多维度 EEG 信息提取和 gated residual fusion 的输入自适应特性.
  \item \textbf{VTF 成功实现了 token-level EEG fusion}, 产生的 enhanced vision tokens 可直接用于下游生成式模型. 虽然 VTF 的分类性能弱于 A2, 但它是通往生成式 EVLM 的关键结构.
  \item \textbf{Route5 Qwen2-VL 是可行的生成式 EVLM 原型}. 它在 strong degradation 条件下的 real EEG class-hit 显著高于 vision-only 和 EEG 控制组, 并能生成自然的 free-form caption. 但生成式模型的 absolute accuracy 仍远低于 A2 分类器.
  \item \textbf{Real EEG 的语义增强作用在退化严重时更明显}. 在 clean 条件下, real-vision gap 仅为 +0.0252; 在 mixed 条件下, 该 gap 扩大到 +0.3255.
  \item \textbf{不应夸大 EEG 的作用}. 本文所有实验均在 40 类的受限分类空间中进行, 生成式 caption 的质量仍远不能与大规模 VLM 在 clean 图像上的 caption 相比. 本文的工作定位是 ``小样本 EEG-enhanced VLM prototype'', 而非 ``脑电解码任意语言''.
\end{enumerate}
