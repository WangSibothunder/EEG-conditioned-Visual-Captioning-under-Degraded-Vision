\section{生成式视觉语言模型设计}

在语义分类任务验证了 EEG 辅助效果之后，本章将 EEG-enhanced vision tokens 接入生成式视觉语言模型，实现从退化图像和 EEG 到自然语言描述的生成通路。

\subsection{生成式描述任务的设计目标}

生成式描述任务的目标是：给定一幅退化图像和配对的 EEG trial，模型生成一段自然语言 caption 描述图像内容。与第 5 章的语义分类任务不同，这里期望模型输出的不是固定的类别标签，而是自由形式的描述文本。

需要说明的是，该任务的定位并不是``从脑电直接读取任意语言''。本课程设计在约 40 类、12000 条 trial 的数据规模下，目标是将 EEG 作为视觉语义增强信号，利用 EEG 携带的关于图像内容的额外神经表征，辅助生成式视觉语言模型在退化条件下产生比纯视觉输入更准确的描述。

\subsection{轻量级 GRU 图像描述基线}

在接入大型生成式模型之前，先实现了一个轻量级的 GRU-based autoregressive caption decoder 作为生成基线。该 decoder 以 VTF 输出的 enhanced vision tokens 的平均池化向量为初始隐状态，使用单层 GRU 逐步解码文本 token。模型参数量约为 2.8M，在 BLIP 伪 caption 上训练。

GRU baseline 能够稳定生成 free-form caption，其优点是训练快速、不需要大型预训练模型、输出始终为自然长度文本。但 GRU decoder 缺乏丰富的语言先验知识，生成描述的语法自然度和概念准确性明显弱于预训练视觉语言模型。GRU baseline 的主要价值在于验证 EEG-enhanced vision tokens 可以驱动自回归 caption 生成。

\subsection{基于 Qwen2-VL 的最终生成方案}

最终采用的生成方案结构如图~\ref{fig:route5} 所示。

核心工作流程如下：(1) 退化图像经 CLIP ViT 得到 visual tokens，与 EEG tokens 经 VTF cross-attention 融合得到增强视觉 token；(2) 增强视觉 token 先通过本设计实现的轻量 prefix adapter 映射到 Qwen2-VL 可接收的嵌入空间，再与语义提示文本共同输入生成式解码器；(3) 在输入中附加语义提示信息（A2 预测的最可能类别）和退化类型标签，以文本 token 形式插入生成 prompt；(4) Qwen2-VL-2B-Instruct 解码器在 teacher-forcing 模式下进行自回归训练，测试时以 ``corruption\_type:'' 为前缀触发生成。

\subsection{LoRA 微调与训练配置}

生成式模型使用 LoRA 在 Qwen2-VL 的 Query 和 Value 投影矩阵上施加低秩适配，秩 $r=8$，缩放因子 $\alpha=16$。可训练参数约 1.3M，远小于模型总参数量。训练时冻结所有原始模型权重。实验还对比了 $r=16$ 的配置。

在训练目标方面，测试了两种策略：(1) 类别名称为目标——直接以类别正确性为优化目标；(2) 类别名称配合过滤后 BLIP 伪 caption 为目标——生成文本更自然，但 BLIP 伪 caption 中可能存在的噪声会影响类别准确性。实验发现，类别目标更有利于提升 caption 类别命中率，类别配合 BLIP 伪 caption 目标更有利于提高文本自然度和有效输出率。本文分别报告两类目标的效果，最终以 Qwen2-VL LoRA $r=8$ 配合多候选重排序作为主要生成式结果。

\subsection{多候选重排序策略}

推理时为每个输入生成 $N=5$ 个候选 caption（通过调整温度参数、top-p 等解码策略），依据候选 caption 的有效性（非空、非 URL/代码）、重复率和长度进行初步过滤，然后计算各候选与 A2/VTF 预测的 top-k 语义概念或 CLIP 文本原型之间的相似度，选取语义一致性最高的候选项作为最终输出。这一过程不依赖测试集真实类别标签，caption 类别命中率仅用于最终评估。多候选重排序在不修改模型参数的情况下明显提升了有效输出率和定性展示效果。
